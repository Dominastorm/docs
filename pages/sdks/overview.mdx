# SDKs

You can interface with Hop's API either over REST, or via libraries for certain languages. Below is a list of our official, supported libraries.

| Name       | Language   | URL                                                                      | Official | Documentation                      |
| ---------- | ---------- | ------------------------------------------------------------------------ | -------- | ---------------------------------- |
| @onehop/client | TypeScript | [npmjs.com/package/@onehop/client](https://www.npmjs.com/package/@onehop/client) | ✅       | [/sdks/client/js](/sdks/client/js) |
| @onehop/react | TypeScript | [npmjs.com/package/@onehop/client](https://www.npmjs.com/package/@onehop/react) | ✅       | [/sdks/client/react](/sdks/client/react) |
| @onehop/js | TypeScript | [npmjs.com/package/@onehop/js](https://www.npmjs.com/package/@onehop/js) | ✅       | [/sdks/server/js](/sdks/server/js) |

## REST API

We don't have SDK support for every language out there - *yet*. So, if you'd like to use our REST API directly, we have documentation on how to use that [here](/reference/rest_api).
